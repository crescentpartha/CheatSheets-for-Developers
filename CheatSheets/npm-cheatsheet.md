# NPM CheatSheet for Developers

## Basic Syntax of Node Package Manager

| Command | Description |
| ------- | ----------- |
| `npm init` | Initialize a node.js project |
| `npm install` | Install package.json dependencies |
| `npm -g uninstall <name> ` | Uninstall global package |
